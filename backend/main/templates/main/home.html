Brain Vibe

Learning Topic Tracking for AI-Generated Code

Project Overview

Brain Vibe monitors AI-generated code changes, identifies new topics to learn, and builds a knowledge graph showing how topics relate to each other.

The system tracks your learning progress as you mark topics "learned" and visualizes your knowledge journey.

Available Endpoints

Project Endpoints

  • /api/projects/ - List and create projects
  • /api/projects/stats/ - Project statistics
  • /api/projects/{id}/analyze_changes/ - Analyze repository changes

Topic Endpoints

  • /api/topics/ - List and create topics
  • /api/topics/{id}/mark_as_learned/ - Mark a topic as learned (DRF action)
  • /api/topics/{id}/prerequisites/ - View topic prerequisites
  • /api/topics/{id}/complete/ - Mark a topic as learned (dedicated endpoint)

Analysis Endpoints

  • /api/analysis/code/ - Analyze code changes (POST)
  • /api/projects/{id}/analyze-diff/ - Analyze Git diffs (POST)

Master Brain

  • /api/master-graph/ - View the consolidated master graph of all topics

Other API Endpoints

  • /api/hello/ - Test endpoint
  • /api/dependencies/ - Topic dependencies (legacy)
  • /api/changes/ - Code changes history

Authentication

  • /admin/ - Django admin interface
  • /api-auth/ - REST framework authentication
